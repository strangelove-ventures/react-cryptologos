## Installing

```sh
# using npm
npm install react-cryptologos

# using yarn
yarn add react-cryptologos
```

## Usage

```jsx
// using commonjs
const { SvgSubstratumSub } = require("react-cryptologos");

// using esm
import { SvgSubstratumSub } from "react-cryptologos";

// example usages
function App() {
  return <SvgSubstratumSub />;
}
```
